Practice & Land

First launch on macOS

Since Landit is distributed outside the App Store, macOS Gatekeeper may block it on first open. To fix this:
Right-click Landit.app → Open — then click "Open" in the dialog. You only need to do this once.
Alternatively, run xattr -cr /Applications/Landit.app in Terminal before opening.

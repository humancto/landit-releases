# Landit

**AI-powered job search & interview prep for macOS**

Landit is a native macOS desktop app that helps you land your dream job with AI-powered tools for every step of the process.

## Features

- **Live AI Interviews** — Practice with a real-time AI interviewer (voice + video) powered by Gemini Live API
- **Job Discovery** — Multi-source job aggregation with AI-powered scoring and matching
- **Interview Prep** — Text-based mock interviews with detailed AI feedback
- **Career Coach** — AI chat assistant for career guidance
- **Resume & Documents** — Import, manage, and analyze your job search documents
- **Application Tracker** — Pipeline management, follow-ups, ghost detection

## Download

Download the latest `.dmg` from the [Releases](https://github.com/humancto/landit-releases/releases) page.

### System Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon or Intel Mac

### Setup

1. Download and open the `.dmg`
2. Drag **Landit.app** to your **Applications** folder
3. Launch Landit and add your AI API key in **Settings** (Gemini recommended for live interviews)
